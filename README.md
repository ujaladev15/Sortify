🔢 Sorting Visualizer

Sorting Visualizer is an interactive web application designed to help users understand and analyze the working of various sorting algorithms through real-time visual representation 📊. The application demonstrates how different algorithms process data step-by-step, making complex concepts easier to grasp for beginners and learners 🎯.

The project provides visual animations for popular sorting algorithms such as Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, and Quick Sort ⚡. Users can control the speed of execution ⏱️, generate random arrays 🔀, and observe comparisons and swaps dynamically 🔄, which enhances conceptual clarity and algorithmic thinking 💡.

The application is built with a strong focus on performance 🚀, responsiveness 📱, and user experience 🎨. It features a clean and intuitive interface that allows users to interact seamlessly while learning sorting techniques in a practical and engaging way 🧠.

✨ Key Features

• Real-time visualization of multiple sorting algorithms 👀
• Adjustable sorting speed for better understanding ⏳
• Dynamic array generation and interactive controls 🎛️
• Responsive and user-friendly interface 📐
• Clear visual representation of comparisons and swaps 🔍

🛠️ Tech Stack
React.js, JavaScript, HTML, CSS
